# VoiceNotes


## About this Project

## Technology stack

- **Front-end:** [React-Native](https://reactnative.dev/)
- **Build:** [Expo](https://expo.dev/)

## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/iamdevmarcos/myNotes.git
```

2. Then go to the project's folder:

```sh
cd voice-notes
```

3. Install all dependencies:

```sh
npm install
```

4. Run locally:

```sh
npm start
```
